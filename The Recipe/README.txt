The Recipe
----------------

**Situation:**

Es bastante común que las personas quieran alcanzar un perfil profesional pero que no sepan que conocimientos necesitan
Es común que no saber cuál puede ser una buena fuente de conocimientos de todas las que se ofrecen en internet.
Es común no tener acceso a personas que ya lograron el objetivo profesional  para orientación acerca de qué puede ser lo siguiente a estudiar.

**Complication:**

Se pierde mucho tiempo en darse cuenta de que es lo siguiente por estudiar
Se pierden oportunidades de crecimiento profesional por desconocer qué es lo que nos falta aprender.

**Preguntas:**

Cómo un sistema puede ayudarme a acceder a las rutas de aprendizaje de otros que consiguieron el mismo objetivo profesional que yo quiero cumplir?

**Respuestas:**

- Se necesita un sistema donde los usuarios puedan compartir las rutas (recetas) de aprendizaje que siguieron para poder crecer profesionalmente.
este sistema se debe basar en la idea de poder compartir las rutas de aprendizaje (recetas) par conseguir un perfil profesional determinado. La plataforma debería permitir que usuarios registrados puedan definir sus propias rutas de aprendizaje, publicarlas y poder hacer copias (forks) a partir de rutas definidas por terceros. 

**Requerimientos:**

- Como usuario, podré crear una cuenta para crear y compartir mis recetas. (38 puntos)

- Como usuario registrado, podré actualizar mi información de mi perfil.(38 puntos)

- Como usuario registrado, podré ver en mi inicio mi receta actual, y sus pasos a seguir.(115 puntos)

- Como usuario registrado, podré ver todas mis recetas actuales y principal. (115 puntos)

- Como usuario registrado, podré buscar otras recetas. (116 puntos)

- Como usuario registrado, podré ver otras recetas y hacer una copia (fork) de una receta existente para mi listado. (154 puntos)

- Como usuario registrado, podré editar una receta de mi listado, y establecer el orden de los pasos a seguir. (154 puntos)

- Como usuario registrado, podré agregar un nuevo paso a una receta existente de mi listado.(154 puntos)

- Como usuario registrado, podré editar un paso de una receta existente de mi listado.(116 puntos)
